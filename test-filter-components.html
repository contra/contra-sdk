That's it! No more coordinating multiple components.

How it works:

1. User clicks FilterFramer button
2. Modal opens with all filter controls
3. User adjusts filters and clicks "Save filters" 
4. FilterFramer dispatches: contra:filterChange { filters: {...} }
5. ExpertListFramer/ExpertGridFramer receive filters and update results
6. Modal closes automatically
