Multi-List SDK Demonstration

This example shows how to implement multiple independent expert lists on a single page:

  • Each list has its own data-contra-list-id and data-contra-program
  • Filters target specific lists using data-contra-list-target
  • Each list maintains independent state, pagination, and filter options
  • All filter options are automatically populated from each program's API
